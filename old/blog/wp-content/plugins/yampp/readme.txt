== Description ==

This plugin will show most popular posts under each posts based on the post views.

== Installation ==

1. Upload the 'YA_wp_popular_posts.php' file to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
